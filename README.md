# Selenium Basics – Java Automation Project

##  About
This project covers the fundamental concepts of Selenium WebDriver using Java.  
It is designed practical examples of browser automation.

##  Topics Covered
- WebDriver setup
- Locators (ID, Name, XPath, CSS Selector)
- Handling Alerts
- Handling Frames & Windows
- Dropdowns
- Implicit & Explicit Waits
- Basic Test Structure

##  Tech Stack
- Java  
- Selenium WebDriver  
- Maven  

##  Requirements
- Java JDK  
- Maven  
- Chrome/Firefox Browser  
- WebDriver (ChromeDriver/GeckoDriver)
